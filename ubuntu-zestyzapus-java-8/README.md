# Ubuntu 17.04 (Zesty Zapus) with java 8

Ubuntu 17.04 with java 8 installed

## Build

```
sudo docker build -t patrickvanamstel/ubuntu-zestyzapus-java-8 .
```

## Publish

```
sudo docker push patrickvanamstel/ubuntu-zestyzapus-java-8
```
